SassScript是什么

In addition to the plain CSS property syntax, Sass supports a small set of extensions called SassScript. SassScript allows properties to use variables, arithmetic, and extra functions. SassScript can be used in any property value.—来自官方文档
译文: 除了纯css语法，sass还有少量的叫SassScript的扩展。在SassScript中属性可以使用变量，
算法，和其他方法。SassScript可用于任何属性值。

    数据类型

    • SassScript有7种主要的类型
    1. numbers(如：1.2,12,12px)
    2. strings of text,有没有引号都可以（如：”foo”,’far’,baz）
    3. colors(如：blue,#044534,rgba(255,255,0,0.5))
    4. booleans(如:true,false)
    5. nulls(如：null)
    6. lists of values,用空格或逗号分隔(如：1.5em 1em 0.3em,Helvetica,Arial,sans-serif)
    7. maps from one to another(如：（key1:value1,key2:value2）)

      lists,一系列值得组合，用逗号或分号隔开，在SassScript中有一些list function

  • nth，可以访问在lists中的第几个项目，nth function
  • join ,将多个lists合并，join function
  • append,添加项目到list中，append function
  • each,为每个项目添加样式，each functin
